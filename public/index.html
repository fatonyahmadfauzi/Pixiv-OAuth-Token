Pixiv OAuth Web

Toolkit untuk mendapatkan token OAuth Pixiv melalui CLI, GUI, maupun Web dengan alur PKCE yang aman dan deployment siap Netlify/Vercel.

PKCE Flow Netlify & Vercel Ready Release Download

Project Overview

Project ini menyediakan helper untuk exchange dan refresh token Pixiv OAuth, plus build script untuk menghasilkan artifact Portable/Setup di Windows dan dukungan Linux binary.

Available Modes

  • CLI: pixiv_login.py
  • GUI: pixiv_login_gui.py
  • Web: static UI + serverless /api/token

Requirements

  • Python 3.11+
  • Dependencies: requests, pyinstaller
  • Windows build toolchain for setup installer
